#1 클라우드 인수인계 메뉴얼

1. 모여락 메인 클라우드 계정에는 매년마다 인수인계 파일(폴더) 누적하기, 회장단을 통해 인수인계
2. 새로운 운영진이 구성되면, 회장단은 메인 클라우드 계정에 폴더 추가(ex. 2026년도 모여락 운영)
3. 새롭게 구성된 운영진들의 구글 계정을 받아서, 새로 추가한 폴더의 접근 권한 부여(보기 및 수정)
4. 다음 운영진이 구성될 시기가 되면, 메인 클라우드 계정에 인수인계 폴더를 만들고 인수인계 내용 작성
5. 1년동안 운영했던 폴더는 필요에 따라 인수인계 폴더로 이동, 또는 개인 소장(평가 시트 등 민감할 수 있는 내용은 알아서 처리)

#2 클라우드 구성

1. 매년 누적되는 인수인계 폴더
2. 악보 폴더: 각 파트장들은, 활동 이후 해당 기수의 활동부원에게서 세션별로 악보를 모아서 악보 폴더에 저장
3. 모여락 회계장부: 매년(또는 학기별로) 총무가 동아리 회계장부 파일을 기록하여 저장

#3 클라우드 페이지에서 확인할 수 있는 내용

1. 모여락 메인 클라우드 계정의 '악보' 폴더에서 보유중인 악보 목록
* 저작권 이슈가 있을 수 있으니 모여락 부원 개인이 직접 악보 폴더에 접근하게끔 하지는 말기
2. 모여락 회계 장부 폴더: 클라우드의 회계 장부 폴더는 일반 부원도 회계 장부를 직접 확인할 수 있도록 '링크가 있는 모든 사용자'에게 '보기' 권한이 부여됨
3. 인수인계 폴더 및 평가 시트 폴더 링크: 어차피 권한 없으면 못 봄, 매번 회장이 운영진 개개인한테 카톡 링크 보내는거보단 편하지 않을까
